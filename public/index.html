📖 Cara Penggunaan
🏠
Masuk Ruangan
  1. Isi nama panggilan kamu
  2. Klik "Buat Ruang Baru" → bagikan kode ke pasangan
  3. Atau isi kode dari pasangan → "Gabung"
📺
Nonton Bareng
  1. Di dalam room, paste link YouTube atau video
  2. Tekan play — otomatis tersinkron dengan pasangan
  3. Pause, skip, volume — semua tersinkron real-time
🖥️
Share Screen
  1. Klik tombol Share Screen di room
  2. Pilih tab, window, atau seluruh layar
  3. Pasangan bisa langsung melihat layarmu
🎮
Game Bareng
  1. Klik tab Game di dalam room
  2. Pilih game: XOX, Sambung 4, Suit, dll
  3. Skor tersimpan otomatis — lihat di scoreboard
💬
Chat Global
  1. Klik menu Chat Global di halaman utama
  2. Ngobrol dengan semua pengunjung Ruang Pacaran
  3. Klik 🎵 Tebak Lagu untuk mulai game tebak lagu
  4. Makin sering chat, badge level naik otomatis
🎵
Tebak Lagu
  1. Di chat global, klik tombol 🎵 Tebak Lagu
  2. Penggalan lagu akan diputar selama 30 detik
  3. Ketik judul lagu di kolom chat secepat mungkin
  4. Yang pertama benar = menang! 🏆
☕
Donasi
  1. Klik "Donasi via Saweria" di halaman utama
  2. Donasi min. Rp 3.000 = efek kembang api untuk semua
  3. Tulis pesan — akan tampil di notifikasi overlay
  4. Namamu masuk ke Top Donatur 🏆
💝
Cari Pacar Online
  1. Dari halaman utama, klik "Cari Pacar Online"
  2. Sistem akan mencocokkan kamu dengan orang lain secara random
  3. Chat, nonton, dan bermain bersama orang baru
